# AI Test Report Analysis Project

This project analyzes automation test results and generates:
- Pass / Fail summary
- Percentage stats
- Graphical charts
- Gemini AI analysis summary
- HTML Report

## How to Run

1. Install dependencies
   pip install -r requirements.txt

2. Add Gemini API Key
   setx GEMINI_API_KEY "YOUR_API_KEY"

3. Run Analysis
   python main.py

4. Report will be generated inside /reports folder
