
AI Automation Testing Framework (API + Web + Allure + GenAI Analysis)

Features:
- Pytest based framework
- API testing using requests
- Web automation using Playwright (placeholder)
- Allure HTML reporting
- Gemini AI analysis integration (template)

Steps:
1. pip install -r requirements.txt
2. pytest --alluredir=allure-results
3. allure serve allure-results
